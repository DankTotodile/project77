Project on Console


Visual Studio Code is an Integrated Development Environment created by Microsoft in 2015. It is an advanced programming software and supports variety of languages like HTML, Css, Javascript, Python, Java, C/C++/C# etc. For more info: https://code.visualstudio.com/docs

A console screen is a screen where the developer can test commands before displaying it on the HTML page. To print something on the console screen, we use the command console.log("text").
